
## Step 1 — Clone Repository

```bash
git clone https://github.com/dummy-account-100/sakthi-production
cd sakthi
```

---

## Step 2 — Install Docker Desktop

Ensure Docker Desktop is:

* Installed
* Running
* WSL2 enabled (if on Windows)

Verify:

```bash
docker --version
docker compose version
```

---

## Step 3 — Build Containers (No Cache)

From project root:

```bash
docker compose build --no-cache
```

---

## Step 4 — Start Containers


```bash
docker compose up -d
```

## Step 6 — Access Application

Open browser:

```
http://localhost
```


To Stop all the Containers use this cmd:
 
docker-compose down
